fv.envoy.directive:envoyList Defines a directive which will display a list of all messages for a form. The template for the list is the property `templateUrl` of $envoyProvider. Must have an `envoyMessages` ancestor.

Usage

as attribute
<ANY >
   ...
</ANY>

Example

<div envoy-list="configForm">
</div>
<!-- or -->
<envoy-list for="configForm"></envoy-list>
