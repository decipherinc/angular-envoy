envoyList
directive in module fv.envoy

Description

Defines a directive which will display a list of all messages for a form.

The template for the list is the property templateUrl of $envoyProvider.

The target form can be specified, by name (with interpolation available), in the envoyList attribute or the for attribute. This attribute may be omitted if the envoyList directive has an envoyMessages ancestor.

Usage

as attribute
<ANY envoy-list>
   ...
</ANY>

Example

<div envoy-list="configForm">
</div>
<!-- or -->
<envoy-list for="configForm"></envoy-list>
