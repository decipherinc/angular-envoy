envoyMessages
directive in module fv.envoy

Description

Enables display of messages for a form.

Usage

as element:
<envoy-messages
       [parent]="{string}">
</envoy-messages>
as attribute
<ANY envoy-messages
     [parent]="{string}">
   ...
</ANY>

Parameters

ParamTypeDetails
[parent]string

If this directive is in a subform of some other form which is also using the envoyMessages directive, and you wish to display messages within its list, specify its form name here.
