form
directive in module fv.envoy

Description

Alias: ngForm

This directive replaces AngularJS' form directive.

Two differences:

  • The controller is replaced with a MessagesFormController.
  • The directive creates a new Scope. See the $alias property for further information.

Usage

as element:
<form
       [name="{string}"]>
</form>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
name
(optional)
string

Name of this form. If omitted, the form is ignored by Envoy.
