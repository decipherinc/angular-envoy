envoyAction
directive in module fv.envoy

Description

Describes a directive wherein you can supply an action (AngularJS expression) to be executed from the message list for a particular control.

In short, you want to use this to activate a form field when the user clicks on the error message.

Usage

as attribute
<ANY envoy-action>
   ...
</ANY>

Example

<input name="title"
       type="text"
       ng-model="myModel.title"
       envoy-action="doSomething()"/>
