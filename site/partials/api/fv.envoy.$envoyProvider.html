$envoyProvider
service in module fv.envoy

Description

Allows configuration of options for Envoy; see the options() method.

Default Options

  • levels: Three (3) default levels. ok, warning, and error, in increasing severity, having descriptions "Fixed!", "Warning", and "Error", respectively.
  • defaultLevel:ok`
  • dataFileUrl: messages.json
  • templateUrl: partials/messages.html
