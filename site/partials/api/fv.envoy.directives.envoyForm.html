envoyForm
directive in module fv.envoy

Description

Use this instead of AngularJS' form (or ngForm) directives.

Main differences:

  • It enables tracking of messages on the form.
  • The controller is replaced with a EnvoyFormController.
  • The directive creates a new Scope. See the $alias property for further information.

Usage

as element:
<envoy-form
       [name="{string}"]>
</envoy-form>
as attribute
<ANY envoy-form
     [name="{string}"]>
   ...
</ANY>
as class
<ANY class="envoy-form [name: {string};]">
   ...
</ANY>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
name
(optional)
string

Name of this form. If omitted, the form is ignored by Envoy.
